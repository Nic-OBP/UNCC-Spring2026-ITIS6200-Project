# UNCC-Spring2026-ITIS6200-Project

This project is intended to demonstrate a method of creating a secure connection for BlueTooth devices with limited IO.

### To run vulnerable simulation:
1: In a terminal; run computer.py\
2: In a second terminal; run attacker.py\
3: In a third terminal; run peripheral.py

### To run secure simulation:
1: In a terminal; run computer.py\
2: In a second terminal; run attacker.py\
3: In a third terminal; run peripheral.py passkey
